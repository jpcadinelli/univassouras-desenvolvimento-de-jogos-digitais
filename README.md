# Desenvolvimento de Jogos Digitais (Univassouras)

Repositório da disciplina **Desenvolvimento de Jogos Digitais**, com materiais e projeto prático criados no **GDevelop** durante as aulas do professor **Fábio**.

## 🎯 Objetivo deste repositório

Este repositório serve para:

- Organizar os arquivos de aula e exercícios práticos;
- Registrar a evolução do projeto ao longo do semestre;
- Facilitar revisão, continuidade dos estudos e versionamento.

## 🧩 Ferramenta utilizada

- [GDevelop 5](https://gdevelop.io/) (engine no-code/low-code para desenvolvimento de jogos 2D)

### Sobre os arquivos

- **aula20260303/aula20260303.json**: arquivo principal do projeto GDevelop.
- **aula20260303/aula20260303.json.autosave**: versão de autosave do editor.
- **aula20260303/New Animation-...piskel**: arte/sprite editada no Piskel.
- **aula20260303/assets/**: imagens usadas no jogo.

## 🕹️ Conteúdo implementado até agora

No estado atual do projeto (aula 2026-03-03):

- Cena principal com personagem e plataformas;
- Objeto **Player** com comportamento de plataforma (movimentação e pulo);
- Objeto **Plataforma** com comportamento de plataforma sólida;
- Câmera suave seguindo o jogador (extensão Smooth Camera);
- Evento para virar o sprite para esquerda/direita com base nas teclas pressionadas.

## 🚀 Como abrir e executar o projeto

1. Instale o GDevelop (desktop ou versão web).
2. Abra o GDevelop.
3. Selecione **Open a project**.
4. Navegue até `aula20260303/aula20260303.json`.
5. Clique em **Preview** para testar o jogo.

## ✅ Boas práticas para a disciplina

- Salvar com frequência e manter o arquivo principal versionado no Git;
- Fazer commits pequenos e descritivos por aula;
- Organizar assets por pasta e com nomes claros;
- Documentar no README o que foi aprendido e implementado a cada aula.

## 🗺️ Sugestão de evolução

Próximos passos recomendados para as próximas aulas:

- Adicionar coleta de moedas/pontos;
- Criar tela de início e tela de game over;
